# 🕵️ Clue Digital Scorecard

A digital scorecard for the Clue board game in Spanish. Track which cards each player has during the game with an intuitive web interface.

## Features

- **Interactive Grid**: Click cells to cycle through states (empty → ✓ → ✗)
- **Multi-Player Support**: Up to 6 players
- **Save/Load Games**: Export and import game states as JSON files
- **Mobile Responsive**: Works on phones, tablets, and desktop
- **Spanish Interface**: Complete localization for Spanish speakers

## Quick Start

1. Open `index.html` in any modern web browser
2. Click on cells to mark cards:
   - ✓ = Player has the card
   - ✗ = Player doesn't have the card
3. Use "Guardar Partida" to save your progress
4. Use "Cargar Partida" to resume a saved game

## Game Categories

- **¿Quién?** (Who?) - 6 characters
- **¿Con qué?** (With what?) - 6 weapons
- **¿Dónde?** (Where?) - 9 locations

## Technical Details

- Pure HTML/CSS/JavaScript - no dependencies
- Client-side file operations
- Responsive design with mobile optimization
- Cross-browser compatible
